## 使用手順

1. input/input.jsonにjsonオブジェクトを入力します。
2. 以下のコマンドを実行します。```python3 generate.py```

###出力先
- output/freezed_output.dartにfreezedのモデルが出力されます。
- output/freezed_output.dartにfreezedのモデルが出力されます。
